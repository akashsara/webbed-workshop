Content

HTML

HTML is a markup language used for creating web pages and web applications. It describes the structure of the webpage semantically. Every website you've ever been to uses HTML. HTML can be considered as the skeleton of the site.

CSS

CSS or Cascading Style Sheets is a language used for describing the appearance of a webpage. Everything from the background color to the font size is managed by CSS. If HTML is the skeleton of the site, CSS makes up the clothes.

JavaScript

JavaScript or JS is a client-side scripting language. It is used to make a web page interactive. It is built-in to every modern browser and does not need to be installed separately. Along with CSS and HTML, it makes up the 3 core technologies of the World Wide Web.

MySQL

MySQL is an open source RDBMS. It is the most widely used database management system in the world. It uses simple SQL commands for all of its tasks.

PHP

PHP is a server-side scripting language usually used in the back-end of the site. It is responsible for collecting form data, generating dynamic page content and even storing and receiving cookies.
